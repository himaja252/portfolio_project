PROJECTS

Here are some of my projects:

  1. Student Achievement Management System
    Implemented a web application capable of storing and showcasing all the extra curricular achievements of the students in the field of sports, cultural activities, technical activities along with course category. In this course category, students can showcase the additional courses taken from various platforms like Udemy, Coursera and NPTEL.
  2. Medicinal and Toxic Plant Classification System
    Developed a plant classification system utilizing MobileNetV2, achieving high accuracy in distinguishing between medicinal and toxic plants. Implemented a web application that classifies plants and provides detailed information about their properties. For medicinal plants, the application specifies how they can be used medicinally. For toxic plants, it identifies which parts are toxic and offers guidelines on treatment for toxicity.
  3. Mixed Dog Breed Identification System
    The Mixed Dog Breed Identification System is a machine learning project that aims to identify mixed dog breeds from photos. This system leverages the power of the InceptionV3 model, a convolutional neural network (CNN) that is widely used for image classification tasks. By analyzing the visual characteristics of a dog in a given photo, the system can predict the breeds present in the dog's lineage. This project provides a valuable tool for pet owners, veterinarians, and animal shelters to understand the genetic makeup of mixed-breed dogs, which can be useful for health, training, and adoption purposes. The use of InceptionV3 ensures high accuracy and reliability in breed identification.
